# 易经占卜 — 应用文档

## 应用概述

基于易经六十四卦的本地占卜应用（PWA / Android APK）。用户输入问题后通过自动（随机模拟硬币）或手动（输入真实抛币结果）方式起卦，应用根据卦象和用户问题生成个性化参考建议。

**核心理念**：纯本地运行，无需网络，无需后端，保护用户隐私。

## 遗留问题

### 功能层面
- [x] ~~无解卦能力~~（v1.2 已实现：体用生克 + 动爻位置 + 白话双轨解读）

- [ ] **缺少互卦**：未计算互卦（234爻和345爻组成的卦），梅花易数体用生克分析缺失

### 技术层面
- [ ] **APK 签名**：当前使用 debug 签名，发布需生成正式 keystore
- [ ] **构建脚本**：仅支持 Unix shell（bash），Windows 原生 cmd/PowerShell 不支持
- [ ] **无测试**：项目无单元测试或集成测试
- [ ] **无障碍**：未进行无障碍测试和优化
- [ ] **国际化**：仅支持简体中文

### UI/UX 层面
- [ ] **启动白屏**：APK 冷启动时有短暂白屏
- [ ] **无卦象动画**：结果页六爻一次性展示，缺少逐爻构建的入场动画
- [ ] **无分享功能**：无法生成卦象卡片图片分享
- [ ] **无音效**：抛币/起卦无音效反馈

## 技术架构

```
Web 单页应用 (HTML/CSS/JS)
├── index.html          — 语义化 HTML 骨架，含设置面板和 AI 解读区域
├── app.js              — UI 状态机 + 历史管理 + 主题管理 (~320行)
├── iching-data.js      — 八卦映射 + 64卦数据 + 起卦算法 + 体用分析 (~1164行)
├── js/
│   ├── settings.js     — 设置管理模块（API Key 等）
│   ├── advice-engine.js— 本地解卦引擎（领域分类 + 卦象解读 + 变爻分析 + 收尾）
│   └── api-client.js   — DeepSeek API 客户端（联网 AI 深度解读）
├── styles.css          — 禅意极简样式 + 深色模式 + AI 解读样式 (~1100行)
├── sw.js               — Service Worker 离线缓存
├── manifest.json       — PWA 清单
├── server.js           — 开发用静态服务器 (Node.js)
└── icon.svg            — 应用图标

Android 壳 (iching-apk/)
├── MainActivity.java   — WebView 全屏加载本地 HTML（已包含网络权限）
├── build-apk.sh        — 一键构建脚本 (aapt2 + javac + d8 + apksigner)
└── debug.keystore      — 调试签名密钥
```

### 起卦算法
- 3 枚铜钱抛 6 次（从下往上排爻）
- 3 正 = 9（老阳 ⚊ → ⚋），2 正 1 反 = 8（少阴 ⚋），1 正 2 反 = 7（少阳 ⚊），0 正 = 6（老阴 ⚋ → ⚊）
- 下三爻 → 下卦，上三爻 → 上卦 → 查 64 卦表
- 老阳/老阴 → 变爻 → 之卦

## 开发指南

```bash
# 本地开发（浏览器）
cd "/d/claude deepseek/占卜" && node server.js
# 浏览器打开 http://localhost:8088

# 构建 APK
bash "/d/claude deepseek/iching-apk/build-apk.sh"

# 安装到手机
adb install -r "/d/claude deepseek/iching-apk/iching.apk"

# 查看崩溃日志
adb logcat -d | grep -A 30 "FATAL EXCEPTION"
```

### 同步 Web 到 APK
修改 `占卜/` 下的文件后，构建 APK 前需要将文件同步到 `iching-apk/app/src/main/assets/www/`。构建脚本 (`build-apk.sh`) 已包含此步骤。
